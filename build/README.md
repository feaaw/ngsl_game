# NGSL Word Rush

重要英単語2800語を楽しく学習できるゲーム形式の学習アプリです。

## 🎮 ゲームの特徴

✨ **NGSL 2800語** - 最重要英単語を網羅
🎯 **クイズ形式** - 4択問題で楽しく学習  
💾 **進捗保存** - LocalStorageで自動保存  
🔥 **コンボシステム** - 連続正解でボーナス点  
⏱️ **タイムアタック** - 60秒間で何問解ける？  
📊 **学習統計** - 習得単語数を確認  

## 🚀 使い方

### ローカルで実行

Webサーバーで実行してください（ファイルプロトコルでは動作しません）:

#### Node.jsの場合

```bash
npx http-server -p 8000
```

#### Pythonの場合

```bash
python -m http.server 8000
```

ブラウザで `http://localhost:8000` を開いてください。

### デプロイ

#### GitHub Pages

1. このフォルダの内容をGitHubリポジトリにプッシュ
2. Settings > Pages > Source で`main`ブランチを選択
3. デプロイ完了！

#### Netlify / Vercel

このフォルダをドラッグ&ドロップするだけでデプロイできます。

## 📁 ファイル構成

- `index.html` - メインHTML
- `game.js` - ゲームロジック
- `style.css` - スタイルシート
- `words.js` - 単語データ（2801語、約500KB）

## 🎓 遊び方

1. **START GAME** でゲーム開始
2. 表示される英単語の意味を4つの選択肢から選ぶ
3. 正解すると**コンボ**が加算され、ボーナス点獲得
4. 5コンボごとに時間が5秒回復
5. 不正解だと時間が5秒減少
6. 時間切れでゲーム終了
7. **MY COLLECTION** で習得した単語を確認

## 📊 スコアリング

- 正解: **100点** + コンボボーナス（コンボ数 × 10）
- コンボ例:
  - 1コンボ: 100点
  - 2コンボ: 120点
  - 3コンボ: 130点
  - 5コンボ: 150点 + 時間回復

## 技術仕様

- **フレームワーク不要** - Vanilla JavaScript
- **データ形式** - JSON（words.js）
- **ストレージ** - LocalStorage
- **レスポンシブ** - モバイル・デスクトップ対応
